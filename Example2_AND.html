CSS Media Query using the AND logical operator

An example of three separate media queries that change the background color according to the device (media type).

/* Mobile styles */
@media screen and (min-width: 0px) and (max-width: 400px) {
body {
background-color: #F25757; /* Grapefruit */ }
}

/* Tablet styles */
@media screen and (min-width: 401px) and (max-width: 960px) {
body {
background-color:#61E8E1; /* Teal or Aqua */ }
}

/* Desktop styles */
@media screen and (min-width: 961px) {
body {
background-color: #F2CD60; /* Orange */ }
}

This media query gathers information about the device used to display the content and
then checks to see if the display has a display width of ( 0 - 400px), (401px - 960px), and (over 961px).
