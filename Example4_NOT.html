Using the NOT logical operator in a CSS Media Query

An example of a media query using the NOT logical expression; which MUST contain a media type and a media feature.

@media not all and (min-width: 900px) {
#itWorked {
font-size: 200px;
color: red;
background-color: gray;}
}
}

If the query returns a TRUE statement the font-size will increase 200%, color changes to red, and the background to gray.

Media Type = ALL
Media Feature - min-width
