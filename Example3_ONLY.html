Using the ONLY logical operator in a CSS Media Query

An example of a media query using the ONLY logical expression; which MUST contain a media type and a media feature.

@media only screen and (color) and (min-width: 500px) {
p {
font-size: 150%;
color: plum; }
}

This media query gathers information about the device used to display, to ensure that it is color compatible or monochrome.
and that the disply width is over 500 pixels.
If the statment is true it will increase the font size by 150% and change the font color to plum.

Media Type = screen
Media Feature - min-width and max-width
